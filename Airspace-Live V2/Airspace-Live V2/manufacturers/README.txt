FLIGHT WALL — MANUFACTURER LOGOS
================================

The type band (between the route and the stats) shows the aircraft
MANUFACTURER. By default it shows a clean wordmark (e.g. BOEING).
Drop a logo image here to replace the wordmark with the real logo.

NAMING (lowercase .png)
-----------------------
    boeing.png
    airbus.png
    embraer.png
    bombardier.png
    atr.png
    cessna.png
    cirrus.png
    piper.png
    beechcraft.png
    gulfstream.png
    dassault.png
    mcdonnelldouglas.png
    pilatus.png

If a manufacturer isn't in this list, the app uses its name as the
filename, lowercase with spaces/punctuation removed (tell me the
manufacturer and I'll give you the exact filename).

FORMAT
------
- PNG, transparent background, ideally a WHITE or light version of the
  logo so it reads on the dark card.
- Landscape logos look best (they're scaled to fit, left-aligned).

If no file is present, the clean wordmark is shown automatically, so
nothing ever breaks.

(Note: the earlier "aircraft" folder for per-type plane images is no
longer used in this slot — the slot now shows the manufacturer. Ask me
if you'd like the aircraft images used somewhere else, like a hero image.)
