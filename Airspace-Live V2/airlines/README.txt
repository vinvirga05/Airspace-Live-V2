FLIGHT WALL — AIRLINE LOGOS
===========================

Airline logos load AUTOMATICALLY from a free CDN (pics.avs.io), so most
carriers already show with no work. Drop a file here only if you want to
override the CDN — e.g. a clean WHITE version for the dark background, or
to cover an airline the CDN is missing.

NAMING (IATA code, UPPERCASE, .png)
-----------------------------------
    DL.png   Delta
    AA.png   American
    UA.png   United
    WN.png   Southwest
    B6.png   JetBlue
    NK.png   Spirit
    F9.png   Frontier
    AS.png   Alaska
    BA.png   British Airways   (etc.)

Precedence:  airlines/<IATA>.png  ->  pics.avs.io CDN  ->  colored initials.

FORMAT
------
- PNG, transparent background, ideally WHITE/light so it reads on the
  dark card. Landscape logos look best (scaled to fit, left-aligned).

Tell me an airline and I'll give you the exact filename if unsure.
